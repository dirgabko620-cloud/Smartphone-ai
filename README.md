# 🤖 Smartphone AI - Advanced AI Website for Termux

Website AI paling canggih, cerdas, dan kece yang berjalan di Termux!

## ✨ Fitur Utama

- 🧠 **Advanced AI Chat** - Powered by Multiple AI Models
- 🖼️ **Image Generation** - Create stunning AI images
- 💻 **Code Generation** - Generate & execute code
- 🎤 **Voice Input/Output** - Speech recognition & synthesis
- 📊 **Data Analysis** - Advanced analytics & visualization
- 🔐 **Secure Authentication** - JWT-based auth system
- ⚡ **Real-time Processing** - WebSocket support
- 📱 **Mobile Optimized** - Works perfect on Termux
- 🎨 **Beautiful UI** - Modern React frontend
- 🚀 **Fast Performance** - Optimized backend

## 🛠️ Tech Stack

**Backend:**
- Python 3.10+
- FastAPI (async framework)
- SQLAlchemy (ORM)
- Ollama (Local AI models)
- JWT Authentication

**Frontend:**
- React 18
- Tailwind CSS
- WebSocket
- TypeScript

**Database:**
- SQLite (development)
- PostgreSQL (production)

**AI Models:**
- Ollama (Local)
- OpenAI API (optional)
- Hugging Face (optional)
